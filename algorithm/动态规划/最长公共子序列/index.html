                        # 最长公共子序列

                        # 问题描述

                        给定两个序列X,YX,YX,Y，且∣X∣=m,∣Y∣=n,m≤n|X| = m, |Y| = n,m \le n∣X∣=m,∣Y∣=n,m≤n

                        X=<x1,x2,…,xm>X=<x_1,x_2,\dots,x_m> X=<x1​,x2​,…,xm​>

                        Y=<y1,y2,…,yn>Y=<y_1,y_2,\dots,y_n> Y=<y1​,y2​,…,yn​>

                        求XXX和YYY的最长公共子序列。

                        # 实例

                        X:ABCBDABX: A \quad \color{red}B \quad \color{red}C \quad \color{red}B \quad D \quad \color{red}A \quad BX:ABCBDAB

                        Y:BDCABAY: \color{red}B \quad D \quad \color{red}C \quad A \quad \color{red}B \quad \color{red}AY:BDCABA

                        LCS:BCBALCS: B \quad C \quad B \quad ALCS:BCBA

                        LCSLCSLCS长度：4

                        # 蛮力算法

                        检查XXX的每个子序列是否在YYY中存在，其中XXX有2m2^m2m个子序列，每次检查YYY中是否存在需要O(n)时间。因此蛮力算法最坏情况下时间复杂度为O(n2m)O(n2^m)O(n2m)。

                        # 子问题界定

                        XXX的终止位置是iii，YYY的终止位置是jjj，由参数i,ji,ji,j界定子问题。

                        设X=<x1,x2,…,xm>,Y=<y1,y2,…,yn>X=<x_1,x_2,\dots,x_m>,Y=<y_1,y_2,\dots,y_n>X=<x1​,x2​,…,xm​>,Y=<y1​,y2​,…,yn​>，Z=<z1,z2,…,zn>Z=<z_1,z_2,\dots,z_n>Z=<z1​,z2​,…,zn​>为XXX和YYY的LCSLCSLCS，那么

                        1. 若xm=yn⇒zk=xm=ynx_m=y_n \Rightarrow z_k=x_m=y_nxm​=yn​⇒zk​=xm​=yn​，且Zk−1Z_{k-1}Zk−1​是Xm−1X_{m-1}Xm−1​与Yn−1Y_{n-1}Yn−1​的LCSLCSLCS

                        2. 若xm≠yn,zk≠xm⇒x_m \ne y_n,z_k \ne x_m \Rightarrowxm​=yn​,zk​=xm​⇒ZZZ是Xm−1X_{m-1}Xm−1​与YYY的LCSLCSLCS

                        3. 若xm≠yn,zk≠yn⇒x_m \ne y_n,z_k \ne y_n \Rightarrowxm​=yn​,zk​=yn​⇒ZZZ是XXX与Yn−1Y_{n-1}Yn−1​的LCSLCSLCS

                        # 优化函数的递推方程

                        XXX子序列：Xi=<x1,x2,…,xi>X_i=<x_1,x_2,\dots,x_i>Xi​=<x1​,x2​,…,xi​>

                        YYY子序列：Yi=<y1,y2,…,yi>Y_i=<y_1,y_2,\dots,y_i>Yi​=<y1​,y2​,…,yi​>

                        C[i,j]C[i,j]C[i,j]：XiX_iXi​与YjY_jYj​的LCSLCSLCS的长度

                        C[i,j]C[i,j]C[i,j]有如下规律：

                        C[i,j]=0 \qquad i=0\mbox{ 或 }j=0

                        C[i,j]=C[i-1,j-1]+1 \qquad i,j > 0 \mbox{ 且 } x_i=y_j

                        C[i,j] = max\{C[i,j-1], C[i-1,j]\} \qquad i,j>0 \mbox{ 且 } x_i \ne y_j

                        # 标记函数

                        标记函数B[i,j]B[i,j]B[i,j]的值有三种情况：

                        • C[i,j]=C[i−1,j−1]+1⇒↖C[i,j] = C[i-1,j-1]+1 \Rightarrow \ \nwarrowC[i,j]=C[i−1,j−1]+1⇒↖
                        • C[i,j]=C[i,j−1]⇒←C[i,j]=C[i,j-1] \Rightarrow \ \leftarrowC[i,j]=C[i,j−1]⇒←
                        • C[i,j]=C[i−1,j]⇒↑C[i,j]=C[i-1,j] \Rightarrow \ \uparrowC[i,j]=C[i−1,j]⇒↑

                        for i from 1 to m

for i from 1 to n
  do C[0, j] = 0

for i from 1 to m
  for j from 1 to n
    if X[i] = Y[j] // 最后一个元素相同
      then C[i, j] = C[i-1, j-1] + 1
      		 B[i, j] = 'left-top'
    else if C[i-1, j] >= C[i, j-1]
      then C[i, j] = C[i-1, j]
      		 B[i, j] = 'top'
    else
      C[i, j] = C[i, j-1]
      B[i, j] = 'left'

  if i == 0 or j == 0

	// 如果最后一个字符相同，直接输出

    then StructureSequence(B, i - 1, j)
  else
